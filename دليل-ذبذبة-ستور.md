# تطبيق TS WALLPAPERS (ذبذبة ستور)

نسخة معدّلة من تطبيق Fullscreen Clock (مفتوح المصدر، GPL v3) بعد إضافة المميزات المطلوبة.

## الملفات
- **`TS-WALLPAPERS-v1.2.apk`** ← التطبيق النهائي الموقّع (ده اللي تركّبه/تبيعه).
- **`source/`** ← الكود المصدري الكامل + مفتاح التوقيع `thabthaba.jks`.
- **`wallpapers-manifest-example.json`** ← مثال لملف قائمة الخلفيات (اللينك).
- اسم التطبيق الظاهر: **TS WALLPAPERS** — اسم الحزمة: `store.thabthaba.clock`.
- مفتاح التوقيع: كلمة السر `thabthaba123` (احتفظ بالمفتاح لأي تحديث مستقبلي).

## المميزات
1. **خلفيات متعددة + تبديل بالسحب** يمين/شمال بأنيميشن دوران لطيف (وأزرار الريموت على TV).
2. **كل الأنواع**: صور (JPG/PNG/WebP)، GIF متحرك، فيديو (MP4...).
3. **مصدر الخلفيات — طريقتين معًا:**
   - **لينك واحد (online):** ملف JSON على سيرفرك، تحدّثه فيتحدّث عند الكل.
   - **مجلد على الجهاز (offline):** أي ملف تحطه في المجلد يظهر تلقائي، أو زر "إضافة صور/فيديو من الجهاز" داخل الإعدادات.
4. **الساعة صغيرة تحت يسار** افتراضيًا + إعدادات للمكان والحجم + إظهار/إخفاء.
5. **12/24 ساعة** + ثواني + تاريخ (في الإعدادات).
6. **كل الإعدادات مفتوحة** بدون شراء، ودعم شاشات 16:9 / Android TV.

## إضافة الخلفيات — الطريقة (أ): لينك واحد
ارفع ملف JSON بالشكل ده على سيرفرك، وحُط رابطه في: الإعدادات ← الخلفيات ← "تحديث الخلفيات من الرابط الآن":
```json
{ "wallpapers": [
  { "type": "image", "url": "https://.../a.jpg" },
  { "type": "gif",   "url": "https://.../b.gif" },
  { "type": "video", "url": "https://.../c.mp4" }
]}
```

## إضافة الخلفيات — الطريقة (ب): من الجهاز
- الأسهل: افتح الإعدادات ← قسم الخلفيات ← زر **"إضافة صور/فيديو من الجهاز"** ← اختر الملفات.
- أو انسخ الملفات يدويًا للمجلد ده (بيظهر مساره داخل الإعدادات):
  `Android/data/store.thabthaba.clock/files/Wallpapers/`

## تثبيت التطبيق (مهم لو ظهر خطأ)
لو التثبيت من الكمبيوتر اترفض برسالة **INSTALL_FAILED_USER_RESTRICTED** (حماية شاومي/MIUI):
1. الإعدادات ← خيارات المطوّرين ← فعّل **"تثبيت عبر USB" (Install via USB)** و **"تصحيح USB (إعدادات الأمان)"** (محتاج تسجيل دخول حساب Mi وشريحة).
2. أو ببساطة: انقل ملف `TS-WALLPAPERS-v1.2.apk` للموبايل ودوس عليه عشان يتثبّت يدوي.

## بناء نسخة جديدة من الكود
من داخل `source` (لازم المسار من غير مسافات):
```
./gradlew :app:assembleStandaloneRelease
```
الناتج: `source/app/build/outputs/apk/standalone/release/fsclock.apk` (موقّع بمفتاح المحل).
