# shots/ — historical dev screenshots (safe to ignore)

Numbered emulator/device screenshots (`01_original.png` … `62_icon.png`) captured while
iteratively developing the fork's features: wallpaper rendering (image/GIF/video),
clock positioning, settings navigation, activation/QR flows, landscape checks, and
release smoke tests. Filenames encode the step being verified.

Nothing references these files — not the build, not CI, not the web deploy
(`.vercelignore` excludes them). They are a visual changelog only. Do not use them as a
source of truth for current UI (many predate later changes); run the app instead.
New debugging screenshots may be added here following the `NN_short_slug.png` pattern.
